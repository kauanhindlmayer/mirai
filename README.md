# Mirai

[![Site](https://img.shields.io/badge/Site-miraihq.com-blue)](https://www.miraihq.com)
[![Documentation](https://img.shields.io/badge/Documentation-docs.miraihq.com-green)](https://docs.miraihq.com)
[![Software Requirements Specification](https://img.shields.io/badge/Software%20Requirements%20Specification-Google%20Docs-yellow)](https://docs.google.com/document/d/17WfJqMz5PNQoFi0NaTzUSNIQouQBlWQumMzeMxhsgA0/edit)

Mirai (Japanese word for "future") is a web-based project management tool that aims to help teams collaborate and manage their projects more effectively.

## Features

- User authentication and organization management
- Project dashboards with metrics and visual insights
- Work items, backlogs, boards, and sprint planning
- Wiki pages for project documentation and knowledge sharing
- Tags, personas, and retrospectives
- Real-time updates using SignalR
